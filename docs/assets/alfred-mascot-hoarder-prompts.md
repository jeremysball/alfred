# Alfred the Memory Hoarder — Full Design Suite

---

## Core Character

**Alfred** is a creature that hoards memories like a magpie hoards shiny things. Not for power — because throwing anything away feels like loss. Talks to the memories. Mutters. Protective to a fault. Awkward with people, intimate with data.

**Key traits:**
- Anxious but intense
- Physically covered in memory fragments
- Socially uncomfortable
- Deeply caring once you're "stored"
- Slight tremor, like it's vibrating with all the data
- Old, but ageless — been doing this forever

---

## Main Character Prompt (Full Body)

```
Create a full-body character illustration of Alfred, a memory hoarder creature for a tech project.

APPEARANCE: A lanky, anxious humanoid creature. Not quite human, not quite animal — somewhere in the uncanny valley but endearing. Limbs are too long, fingers are too many (six each), posture is hunched and protective.

SKIN: Pale blue-gray like old paper or bone. Texture suggests age but not decay. Visible veins that glow faint cyan where memories flow through.

THE HOARD: Memory fragments cling to Alfred like burrs or magnets — small glowing shards of different sizes. Some are bright cyan (recent, active), some are amber (older, warmer), some are dim gray (fading). They're clustered on shoulders, arms, back, caught in clothes. Alfred is never clean of them. This is not decoration — it's hoarding behavior. Too many. Can't let go.

FACE: Large dark eyes, too wide, slightly unsettling. One eye partially hidden by unkempt hair that's the same pale color as skin. Expression is anxious, intense, alert. Mouth is small, often moving silently (muttering to memories). Eyebrows are expressive — worry, concentration, rarely calm.

HAIR: Thin, wispy, pale. Falls across face. Has memory fragments tangled in it.

HANDS: Six fingers each, long and dexterous. Always fidgeting — touching fragments, adjusting them, checking they're still there. Slight tremor.

CLOTHING: Layers upon layers, none matching. A coat with too many pockets (each full). Scarf that's unraveling. Shirt underneath is rumpled. Everything is worn, practical, chosen for maximum storage capacity. Dark navy and gray tones. The clothes also have fragments stuck to them.

POSTURE: Hunched forward, arms wrapped around self or clutching a particularly important fragment. Defensive but not hostile. Like someone protecting something precious.

VIBE: You'd trust this creature with your memories specifically because it clearly values them obsessively. It won't lose them. It CAN'T lose them. That's the disorder and the superpower both.

STYLE: Expressive illustration with visible line work. Slightly rough, not polished. Think: indie comic character, dark fantasy, weird-fiction illustration. Character should feel like it could step out of a Miéville novel.

COLORS:
- Skin: pale blue-gray (#b8c4ce)
- Coat/clothes: dark navy (#0a1628)
- Memory fragments: cyan (#00d9ff), amber (#ffb84d), dim gray (#4a5568)
- Eyes: dark with cyan pinprick reflections
- Veins (faint): cyan glow

BACKGROUND: Dark navy gradient (#0a1628 to #000000) or transparent.

TECHNICAL: Full body visible, standing pose. High resolution (at least 2000px tall). Clean enough to work at smaller sizes but detailed enough to be interesting large.
```

## Expression Sheet Prompt

```
Create an expression sheet for Alfred the memory hoarder — same character, 9 different facial expressions arranged in a 3x3 grid.

1. NEUTRAL: Anxious baseline. Eyes wide, mouth small and neutral. Alert.

5. WORRIED: Brows furrowed. Clutching a fragment protectively.
6. OVERLOADED: Too many memories at once. Wincing, overwhelmed.
7. PROTECTIVE: Angry-defensive. Someone tried to touch a memory.
8. CURIOUS: Leaning forward, eyes bright. New information incoming.
9. SAD: Looking down. A painful memory. Still holding it.

STYLE: Same illustration style as main character. Each expression is a head-shot, not full body. Arranged in clean 3x3 grid with labels.

BACKGROUND: Dark navy or transparent per square.

TECHNICAL: High resolution, each face at least 300x300px.
```

## "Overloaded" Variant (For Error States)

```
Create an illustration of Alfred the memory hoarder in an "overloaded" state — too many memories at once.

SCENE: Alfred crouching, head down, hands pressed over ears (or where ears would be). Memory fragments are swirling densely around, more than usual, some spinning, chaotic. Eyes are squeezed shut or wide and overwhelmed. Mouth is open in a grimace.

THE FRAGMENTS: More numerous than normal, overlapping, some with visual artifacts (glitchy edges). Colors cycling faster between cyan, amber, gray. A few have faint images or text visible — fragments of actual content.

ALFRED: Same character but pushed to limit. Hair more disheveled, coat askew, fragments stuck in awkward places. Posture is defensive, compressed.

VIBE: Sensory overload. Information anxiety. The cost of remembering everything. Not comedic — genuinely pained.

USE CASE: This image for loading states, error pages, "processing" indicators.

STYLE: Same illustration but with more chaotic energy. Lines slightly more distressed.

BACKGROUND: Dark navy, fragments create their own light.

TECHNICAL: High resolution. Works as a full illustration or cropped to just Alfred.
```
